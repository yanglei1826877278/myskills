# 我的技能集

个人技能仓库，存放为 Claude Code 编写的自定义技能。

## 技能列表

### 图片处理器
功能全面的图片处理工具。

**主要功能：**
- 格式转换（支持 JPG/PNG/WebP/ICO/BMP/TIFF 互转）
- 分辨率调整
- 图片压缩

**使用方法：**
```bash
/image-processor <文件路径> [选项]
```

详细使用说明请查看 [图片处理器文档](skills/image-processor/SKILL.md)。

## 在 Claude Code 中使用

如需使用这些技能，请将 `skills/` 目录复制到你的 Claude Code 配置目录中。
